## The CRAP Score

- [cargo-crap](https://github.com/Boehs/cargo-crap) - Rust implementation that inspired go-crap
- [gocyclo](https://github.com/fzipp/gocyclo) - Cyclomatic complexity tool for Go (BSD-3-Clause), adapted for `internal/complexity`
- [test-finder](https://github.com/padiazg/test-finder) - Test discovery tool (MIT), adapted for `internal/coverage`

## Metrics

- [Cyclomatic Complexity](https://en.wikipedia.org/wiki/Cyclomatic_complexity) - software complexity metric by Thomas J. McCabe Sr.
- [CRAP Score](https://blog.plataformatec.com.br/2014/10/crap-score-what-and-how-it-works.html) - Cyclomatic Readability and Predictability score
